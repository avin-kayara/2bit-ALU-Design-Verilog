# 2bit-ALU-Design-Verilog
Designed a 2-bit ALU in Verilog supporting 5 operations: ADD, SUB, AND, OR, and XOR, controlled by a 3-bit opcode. The ALU detects carry and underflow conditions. Simulated using Icarus Verilog and verified with GTKWave waveform analysis. All operations produce correct outputs as validated by the testbench.
